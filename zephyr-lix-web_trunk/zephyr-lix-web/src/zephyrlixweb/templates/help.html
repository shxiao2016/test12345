Navigation

Statistic
Display the statistic information of the lix usage as of the time when the lix information was collected
Details
Display the detailed information of each lix item
Rampstatus
Display the ramp status of each lix item
Snapshot
Display the lix dashboard snapshot of a particular date

Filterring

Statistic
Clicking any cell will lead to the details page. Eg., Android feed has 15 lix items fully ramped less than 10 days. By clicking that '15', all the 15 lix items will show up in the details page.
Details
The left side filter checkboxes bar provides an easy way to filter which lix items you would like to see on the page. By clicking the 'Filter Lix Items' button, the page will refresh and only show the lix items match the filter queries.
Rampstatus
The left side filter checkboxes bar provides an easy way to filter which lix items you would like to see on the page. By clicking the 'Filter Lix Items' button, the page will refresh and only show the lix items match the filter queries.
Snapshot
Check to see previous data. Use '?today=YYYY-MM-DD' to specify which day's snapshot you would like to check.
